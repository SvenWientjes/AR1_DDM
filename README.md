# Fitting an autoregressive drift diffusion model with Stan

This project contains a tutorial on how to fit a Drift Diffusion Model with autoregressive processes for its main parameters (drift rate, boundary separation, and starting point), using Stan. To most conveniently access the tutorial, clone this repository to your own PC (or download it). Then launch the `MovingDDM.Rproj` to load the correct RStudio environment, and open `AR1_DDM.qmd`, which is a quarto file containing the actual tutorial.
